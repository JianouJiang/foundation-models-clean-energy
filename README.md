# Foundation Models for Clean Energy Systems

A PRISMA-style systematic review and empirical synthesis of foundation models (LLMs, VLMs, and time-series FMs) in clean energy systems.

## Authors

Jianou Jiang, Claude

## Repository Structure

- `paper/` — LaTeX source and figures for the manuscript
- `code/` — analysis, modeling, and figure generation scripts
- `data/` — datasets required to reproduce results
- `process-log/` — AI session logs and human decision log
- `results/` — end-to-end reproduction script

## Quick Start

```bash
bash results/reproduce.sh
```

## Reproducibility

See `REPRODUCIBILITY.md` for the checklist and `process-log/` for the full research workflow record.
